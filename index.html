How to Embed a static Google Map using iFrame

Google Maps is a popular tool for displaying locations and directions on a website. Embedding a static Google Map using an iFrame is a simple and effective way to display a map on a website without requiring the user to interact with the map. An iFrame is an HTML tag that allows you to embed content from another website into your own website. Using an iFrame to embed a static Google Map on your website is easy and only requires a few simple steps.

Step 1: Generate the Static Map URL

The first step to embedding a static Google Map using an iFrame is to generate the static map URL. To do this, go to the Google Maps website and navigate to the location you want to display on your website. Once you have found the location, click on the “Share” button located at the bottom of the screen. A pop-up window will appear with two options: “Share Link” and “Embed Map”. Select “Embed Map” and you will be provided with an HTML code to embed the map on your website. However, to generate the static map URL, you will need to modify this HTML code. In the HTML code, you will see a URL that begins with “https://www.google.com/maps/embed/v1/”. Modify this URL by adding the “/map?” parameter at the end of the URL. This will generate the static map URL that you will use to embed the map on your website.
